Smart Retail Intelligence Dashboard

Overview

This dashboard analyzes retail performance across stores, months, and holidays using Power BI.

Features

* KPI tracking (Revenue, Growth, Stores)
* Monthly revenue trends
* Store-level performance analysis
* Holiday impact on sales
* Interactive filters (Year, Month, Holiday)


Insights

* Identifies top-performing stores
* Highlights seasonal trends
* Compares holiday vs non-holiday sales

 Tools Used

* Power BI
* DAX
* Data Modeling
